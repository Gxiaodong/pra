This package contains basic data structures (e.g. map, vector, set) and all generalized operations on them (e.g. Vector.toSet()).

Name convention: Classes stating with T are template classes with unspesified content class type(s). Generally I=Integer, D=Double, S=String, X=unspecified

Interfaces: Containers can do batch operations defined in @Interface to all its elements

conventions: Containers will not load from a column of a text table. This is handled by enumerators
